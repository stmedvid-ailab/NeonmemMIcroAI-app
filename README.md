# Neonmem Micro-AI

**Small, stateful AIs whose knowledge lives in a portable memory cartridge — not
in billions of weights.**

A micro-AI is like a developer who finished the courses and has all the project
docs in front of them: a focused specialist that helps with the real, everyday
work — reading logs, recalling decisions, advising, reminding — **privately,
offline, and on hardware you already own.** One knows one field; an orchestrator
coordinates many for bigger tasks.

Small experts + smart coordination, instead of one enormous generalist.

> **Status: early.** See **[docs/VISION.md](docs/VISION.md)**.
> Part of the [Neonmem](https://neonmem.com) family.

## Principles
Private & offline by default · auditable answers · public-domain foundations ·
prove it small, then scale.
